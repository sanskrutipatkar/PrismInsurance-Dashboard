# PrismInsurance- Dashboard
This project features an interactive Power BI dashboard for insurance data analysis, providing insights into premium amounts, coverage, claims, policy status, and customer demographics. The dashboard supports drill-down analysis through dynamic filters and visual breakdowns by policy type, claim status, and age group.
